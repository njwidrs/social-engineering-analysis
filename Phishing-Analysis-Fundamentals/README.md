## 1. Introduction
Phishing is a social engineering attack where attackers attempt to trick users into revealing sensitive information such as credentials or financial details. This lab introduces the fundamentals of phishing email analysis.

## 2. Objective
- Understand common phishing techniques
- Identify indicators of phishing emails
- Learn how attackers manipulate victims through email content

## 3. Platform
- TryHackMe

## 4. Tools Used
- Web browser
- TryHackMe virtual lab

## 5. Steps Performed
1. Logged into TryHackMe
2. Opened the "Phishing Analysis Fundamentals" room
3. Studied phishing components such as:
   - Sender address
   - Email subject
   - Embedded links
   - Attachments
4. Completed questions related to phishing indicators
5. Submitted answers to obtain completion

## 6. Key Phishing Indicators Identified
- Suspicious sender email address
- Urgent or threatening language
- Mismatched URLs
- Unexpected attachments
